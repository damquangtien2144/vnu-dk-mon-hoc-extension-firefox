## 🎯 Giới thiệu

Mỗi mùa ĐKMH, sinh viên VNU thường phải dùng **Ctrl+F** để tìm từng môn học trên trang đăng ký — rất chậm và dễ bỏ sót. Extension này giải quyết vấn đề đó bằng cách:

- **Nhập sẵn danh sách** môn học cần đăng ký
- **Tô sáng (highlight)** tất cả lên trang web chỉ với 1 click
- **Mỗi môn 1 màu** riêng, dễ phân biệt
- **Điều hướng** nhanh giữa các kết quả tìm được

> 💡 Hoàn toàn miễn phí • Không thu thập dữ liệu • Chạy offline

---

## ✨ Tính năng

### 🔍 Tìm kiếm & Highlight
- Nhập thông tin môn học vào 5 ô mặc định: **Tên môn**, **Mã lớp**, **Thứ**, **Ca**, **Giảng đường**
- Dùng dấu phẩy `,` để tìm nhiều giá trị cùng lúc (VD: `Thứ 2, Thứ 4`)
- Nhấn **🔍** hoặc **Enter** để tô sáng môn trên trang web
- Chọn **màu highlight** riêng cho mỗi môn

### ⚡ Quét tất cả (Scan All)
- Highlight **toàn bộ** danh sách lên trang web chỉ với 1 click
- Mỗi môn được tô màu riêng biệt
- Phím tắt: `Ctrl + Shift + S`

### ▲▼ Điều hướng kết quả
- **Badge** hiển thị số kết quả tìm thấy (VD: `1/3`)
- Nút **▲ ▼** nhảy giữa các kết quả trên trang (scroll mượt)
- **Ctrl + ↑↓** điều hướng toàn cục trên trang web (qua tất cả môn)

### ⭐ Nhấp nháy Highlight
- Bấm **⭐** để bật chế độ nhấp nháy cho một môn
- Giúp dễ phân biệt trong nhiều highlight cùng lúc

### 🔄 Kéo thả & Sắp xếp
- Kéo biểu tượng **⠿** để thay đổi thứ tự các môn
- Drag & Drop mượt mà với animation

### 🏷️ Tag ưu tiên
- Gắn tag cho mỗi môn: **Bắt buộc** (đỏ) · **Tự chọn** (xanh) · **Backup** (vàng)
- Giúp phân loại và quản lý danh sách

### 📋 Nhân bản (Duplicate)
- Nhấn **📋** để nhân bản nhanh một môn học
- Tiện cho việc tạo nhiều nhóm lớp cùng một môn

### 🔛 Bật / Tắt
- **Toggle** bật/tắt từng môn (không cần xóa)
- **Toggle All** bật/tắt toàn bộ danh sách

### 🔎 Lọc nhanh
- Ô tìm kiếm ở trên danh sách để lọc nhanh theo tên/mã

### 📁 Import / Export CSV
- **Tải file mẫu** CSV để điền sẵn danh sách
- **Tải lên CSV** để import hàng loạt
- **Xuất CSV** để lưu lại hoặc gửi bạn bè

### 📱 Chia sẻ QR Code
- Tạo **mã QR** chứa danh sách môn học
- Bạn bè quét hoặc **dán mã chia sẻ** để import nhanh

### 📌 Cửa sổ nổi (Floating Panel)
- Chuyển extension thành **cửa sổ nổi** trên trang web
- Vừa xem trang ĐKMH vừa tìm kiếm
- **Kéo** header để di chuyển
- **Kéo viền** để resize (8 hướng)
- **Ghim/Bỏ ghim** để giữ cửa sổ khi click ra ngoài

### ⚙️ Tùy chỉnh cài đặt
- **Đổi tên** nhãn các ô nhập liệu
- **Thêm trường tìm kiếm** mới tùy ý
- **Kiểu highlight**: tô nền · gạch chân lượn sóng · gạch dưới đậm · in đậm + đổi màu
- **Điều chỉnh độ đậm** (opacity) của highlight
- **Xem trước** kiểu highlight trước khi lưu

### 🌐 Song ngữ (Tiếng Việt / English)
- Chuyển đổi ngôn ngữ bằng nút **🌐** trên header
- Toàn bộ giao diện & onboarding đều được dịch

### 🔥 Streak
- Đếm số ngày liên tiếp sử dụng extension
- Động lực cho mùa ĐKMH!

### 🐱 Meme Easter Egg
- Nhấn **🐱 Meme** để bật/tắt meme GIF nhảy nhót

### 📚 Hướng dẫn người dùng mới (Onboarding)
- **10 bước hướng dẫn** chi tiết, bao phủ toàn bộ tính năng
- Full-screen slides với animation premium
- Điều hướng bằng nút hoặc phím **← →**
- Click dots để nhảy thẳng đến bước bất kỳ
- Tự động hiện lần đầu sử dụng

## 📥 Cài đặt

### Cách 1: Từ mã nguồn (Developer Mode)

1. **Tải mã nguồn** về máy (Clone hoặc Download ZIP)
   ```bash
   git clone https://github.com/user/vnu-dk-mon-hoc-extension.git
   ```
2. Mở **Firefox** → nhập `about:debugging#/runtime/this-firefox` vào thanh địa chỉ
3. Nhấn nút **Load Temporary Add-on...**
4. Trỏ tới file `manifest.json` trong thư mục `vnu-dk-mon-hoc-extension-firefox`
5. Extension sẽ xuất hiện trên thanh công cụ ✅

### Cách 2: Từ file XPI (nếu có)
1. Kéo thả file `.xpi` vào trang `about:addons`
2. Xác nhận cài đặt

## 📖 Hướng dẫn sử dụng

### Bước 1: Mở extension
- Click icon 🐹 trên thanh công cụ Firefox
- Hoặc nhấn **Ctrl + Shift + F**

### Bước 2: Nhập danh sách môn học
- Điền thông tin vào các ô: Tên môn, Mã lớp, Thứ, Ca, Giảng đường
- Nhấn **＋ Thêm** để thêm môn mới
- Hoặc **Tải lên CSV** để import hàng loạt

### Bước 3: Tìm kiếm
- Mở trang đăng ký môn học VNU trên tab hiện tại
- Nhấn **⚡ Quét tất cả** (`Ctrl+Shift+S`) để highlight toàn bộ
- Hoặc nhấn **🔍** từng môn riêng lẻ

### Bước 4: Điều hướng
- Dùng nút **▲ ▼** để nhảy giữa các kết quả
- Hoặc dùng **Ctrl + ↑↓** trên trang web

### Mẹo nâng cao
- Dùng dấu phẩy `,` để tìm nhiều giá trị: `MAT1092, MAT1093`
- Chuyển sang **📌 Cửa sổ nổi** để vừa xem vừa tìm
- Gắn **Tag ưu tiên** để phân loại bắt buộc / tự chọn / backup
- **Chia sẻ QR** cho bạn bè cùng lớp

## 🛠️ Công nghệ

| Thành phần | Công nghệ |
|------------|-----------|
| Platform | Firefox Extension Manifest V3 |
| Frontend | HTML5, Vanilla CSS, Vanilla JS |
| Font | [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts) |
| QR Code | [qrcode.js](https://github.com/davidshimjs/qrcodejs) |
| Storage | `browser.storage.local` (hoặc `chrome.storage.local`) |
| i18n | Custom JSON-based (vi/en) |
| Content Script | DOM manipulation, `MutationObserver`-free highlighting |

### Kiến trúc

```
┌─────────────────────┐     chrome.tabs.sendMessage     ┌──────────────────┐
│    popup.js          │ ──────────────────────────────▶ │   content.js     │
│  (Giao diện chính)   │                                 │  (Trang web)     │
│                      │ ◀────────────────────────────── │                  │
│  • Quản lý danh sách │     sendResponse (match count)  │  • Highlight DOM │
│  • CSV/QR/Settings   │                                 │  • Navigate      │
│  • Onboarding        │                                 │  • Floating Panel│
└─────────────────────┘                                  └──────────────────┘
        │                                                        ▲
        │              chrome.scripting.executeScript             │
        └────────────────────────────────────────────────────────┘
```

- **popup.js** quản lý UI, lưu trữ dữ liệu, gửi lệnh highlight
- **content.js** chạy trên trang web, nhận lệnh, thực hiện highlight DOM, tạo floating panel (iframe)
- Floating panel: là `popup.html` được nhúng dưới dạng **iframe** trong trang web, giao tiếp qua `postMessage`
